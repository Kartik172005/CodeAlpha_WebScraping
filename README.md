# CodeAlpha_WebScraping

## Project Title
Books to Scrape Web Scraper

## Objective
This project extracts book information from the "Books to Scrape" website using Python and BeautifulSoup.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Extracted Data
- Book Title
- Price
- Rating
- Availability
- Product URL

## Output
The scraped data is saved as a CSV file named `books_dataset.csv`.
